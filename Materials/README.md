### After Download the DSCH2 package of Zip file, 
- Extract the Zip file ``DSCH_2.zip``
- Go to ```DSCH_2/EXPORT_DSCH_2```
- Run ```Dsch2.exe```

### For MICROWIND_2,
- Extract the Zip file ``MICROWIND_2.zip``
- Go to ``MICROWIND_2/examples/1``
- Run ``MicroWind.exe``

### For Microwind 3.5 [PreInstalled]
- Extract The Zip file ``Microwind 3.5 PreInstalled.zip``
- Go to ``Microwind 3.5/DSCH3.5/system/``
- Run ``Dsch35.exe`` and DSCH version3.5 will start running
- Same as MIcrowind was pre installed.
- Go to ``/Microwind 3.5/Microwind3.5/system/``
- Run ``Microwind35.exe`` and Microwind version3.5 will start running

### For Microwind 3.5 full version .exe
- Download the ``microwindlite35.exe``
- Run the installer ``microwindlite35.exe``
- It will automatically install everything
- Go to ``C:\Program Files (x86)/Microwind 3.5/Microwind3.5/system/``
- Run ``Microwind35.exe`` and Microwind version3.5 will start running
- Same as DSCH 3.5 will be there in ``C:\Program Files (x86)/Microwind 3.5/DSCH3.5/system/``
