## About

This repository is dedicated to organizing all course-related materials for the VLSI Design Lab (CSE4218). It includes:

- Step-by-step lab tasks
- Detailed lab reports
- Reference notes and slides
- Software and tool files required for practical exercises
- Useful external resources for deeper study

The goal is to centralize all VLSI lab work in one place for easy access and future reference.

## Lab Tasks

Each lab session includes:

- A dedicated folder (e.g., `Lab-01`, `Lab-02`)
- Instructions and task descriptions
- Source files submitted
- A formal lab report (PDF) documenting goals, process, results, and conclusions

You can explore these folders to review individual lab work.
